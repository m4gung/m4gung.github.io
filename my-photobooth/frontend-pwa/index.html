Petunjuk Penggunaan

  1. Sambungkan semua perangkat ke WiFi yang sama
  2. Jalankan server di laptop: cd backend-server && npm start
  3. Buka http://localhost:3000 di browser
  4. Pilih Mode Kamera di iPhone
  5. Pilih Mode Preview di iPad
  6. Tekan tombol "Jepret" di iPad untuk mengambil foto
