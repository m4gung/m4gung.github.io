Aplikasi Penjualan

Cara penggunaan Auto-Save (Recommended):
1. Buka file gs/Code.gs di folder penjualan
2. Copy kode ke script.google.com
3. Deploy as Web App → Execute as: Me → Who has access: Anyone
4. Copy URL (结尾 /exec) ke field "Google Apps Script URL" di atas
5. Klik Simpan → Data akan otomatis tersimpan saat online!

Alternatif (tanpa Auto-Save):
- Mode CSV: Sheet harus Public
- Mode API: Butuh Google Cloud API Key
